# Atlas — Mobile UI kit

Mobile recreation of Atlas. Four phone screens shown side-by-side inside a single HTML file. Capture-first interactions.

Screens: **Inbox feed**, **Quick capture with voice**, **Note reader (RTL)**, **Board view**.

Files:
- `index.html` — four iOS device frames
- `screens.jsx` — all four screens in one file (Inbox, QuickCapture, Note, Board)
- `ios-frame.jsx` — iOS device frame starter component
